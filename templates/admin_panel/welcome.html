API Endpoints

  • 2. api/auth/login/ [name='api_login'] (POST for JWT)
  • 3. api/auth/refresh/ [name='refresh'] (POST for token refresh)
  • 4. api/tasks/ [name='task-list'] (GET list)
  • 5. api/tasks/<int:id>/ [name='task-update'] (e.g., id=1; PUT for update)
  • 6. api/tasks/<int:id>/report/ [name='task-report'] (e.g., id=1; GET for report)
